# Astrophage ABM

## Summary

Astrophage ABM is an Agent-Based Model simulation written in Java and JavaFX. Astrophages move on a map, absorb light energy, reproduce, age and die. Light is generated by cores, that create light energy gradients which astrophages can absorb.

---

## Quick Start

Open a terminal in the Programowanie_Obiektowe_Projekt directory and type:

gradle run

Then enter the simulation parameters.

---

## Input Parameters

* Astrophage population
* Board size
* Movement energy cost
* Simulation duration
* Astrophage lifespan
* Number of light cores

---

## Sample Run

Enter the starting amount of astrophage:
100

Enter the size of the board:
50

Enter the energy cost of each movement:
0.5

Enter the length of the simulation:
200

Enter the lifespan of each astrophage:
20

Enter the core amount:
5
